API INTEGRATION
================================================================================

Using the API Client:

   import { apiClient } from '@/lib/api'

   // GET request
   const data = await apiClient.get('/api/national-overview')

   // POST request
   const result = await apiClient.post('/api/analysis', {
     state: 'Maharashtra'
   })

   // Error handling
   try {
     const data = await apiClient.get('/api/endpoint')
   } catch (error) {
     console.error('API Error:', error)
   }

Available Endpoints:
   /api/status                    - Health check
   /api/national-overview         - National statistics
   /api/enrollment-timeline       - Timeline data
   /api/state-distribution        - State data
   /api/explorer/enrollment       - Enrollment data
   /api/analytics/anomalies       - Anomaly detection
   /api/mobility/migration-patterns - Migration data
   /api/policy/recommendations    - Policy suggestions

See lib/constants.ts for all available endpoints.

================================================================================
TROUBLESHOOTING
================================================================================

Problem: Port 3000 already in use
Solution: npm run dev -- -p 3001

Problem: Dependencies not installing
Solution: rm -rf node_modules && npm install

Problem: TypeScript errors
Solution: npm run type-check

Problem: Styles not applying
Solution: rm -rf .next && npm run build

Problem: Changes not reflecting
Solution: Restart dev server (Ctrl+C, then npm run dev)
