================================================================================
GETTING STARTED
================================================================================

1. INSTALLATION:
   cd samvidhan-ui
   npm install

2. SETUP ENVIRONMENT:
   cp .env.example .env.local
   Edit .env.local with your API base URL

3. START DEVELOPMENT:
   npm run dev
   Open http://localhost:3000

4. BUILD FOR PRODUCTION:
   npm run build
   npm start
